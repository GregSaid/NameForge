## üì¶ About the Project

7Miner was created to make it easier to search for unique and rare Minecraft usernames. Whether to reclaim a nostalgic name, grab a common word, or create something aesthetic, 7Miner makes the process simple and efficient. No need to check one by one anymore!

- ‚úîÔ∏è Uses proxies for anonymity and to avoid rate-limits.  
- ‚úîÔ∏è Supports automatic name generation or custom name lists.  
- ‚úîÔ∏è Smart cache that avoids duplicate checks.  
- ‚úîÔ∏è Fully free and open-source.  

---

## üöÄ Features

- ‚úÖ Checks name availability directly via Mojang's official API.  
- ‚úÖ Automatically generates usernames using filters.  
- ‚úÖ Reads names from a custom `names.txt` list.  
- ‚úÖ Local cache to save requests (`debug/cache.txt`).  
- ‚úÖ Discord integration via Webhook (`webhook.txt`).  
- ‚úÖ Supports HTTP/S proxies (`proxies.txt`).  
- ‚úÖ Interactive system with real-time logging.

## üì° Discord Webhook (Optional)

If you want to receive notifications directly in a Discord channel:

2. Paste the URL into the file `webhook.txt`.  
3. Done! All available usernames found will be sent automatically to your Discord channel.
